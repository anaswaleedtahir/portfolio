# Use Next.js with static export

Use the Next.js App Router with TypeScript and static export even though the initial portfolio has one view. Vite would produce a smaller conceptual surface, but Next.js provides prerendered identity content, structured metadata, and a clean path to future writing or case-study routes without requiring a server for the current site; the animated field remains an isolated client component.

## Consequences

- Do not enable experimental React View Transitions while the site has no internal navigation.
- Keep all identity content server-rendered and restrict client JavaScript to the clock, entrance reveal, and ambient canvas.
